Console functions

There are mainly 10 console functions. They are

  1. log(): It is used to log the output to the console
  2. error(): It is used to log error messages to the console
  3. warn(): It is used to give warning message
  4. clear(): Clears the console
  5. time(): Shows current time
  6. timeEnd(): Shows the running time
  7. table(): This method allows us to generate a table inside a console.
  8. count(): This method is used to count the number that the function hit by this counting method.
  9. group(): group() methods of the console object allows us to group contents in a separate block, which will be indented.
  10. groupEnd(): groupEnd() methods of the console object allows us to group contents in a separate block, which will be indented.

Difference between var, let and const

The main difference is var is global scoped whereas let is local scope. If we declare an element using var or let we can reassign them in the same file more than once. But if we declare a variable using const we cannot be able to change the value of the variable.

Datatypes in JavaScript

  1. Strings: A string is a series of characters. Strings are written with quotes.
  2. Numbers: JavaScript has only one type of numbers. Numbers can be written with, or without decimals:
  3. Booleans: Booleans can only have two values: true or false.
  4. Arrays: Arrays are written with square brackets. Array items are separated by commas.
